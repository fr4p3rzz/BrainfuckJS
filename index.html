How to brainfuck:

Symbol Usage
+ Increase by 1 the value of the selected byte
- Decrease by 1 the value of the selected byte
> Move to the next byte in the stack
< Move to the previous byte in the stack
[ ] Start / end cycle: code inside brackets will be executed X times based on the byte on the current position.

infinite loops and stack overflows may lead to freeze or crash.
. Print in the output the value of current byte (ASCII encoded)
, Insert a keyboard value (ASCII encoded)
